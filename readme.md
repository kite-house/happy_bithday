# Тестовое задание Telegram-бот на Python
## Цель: Проверить знание и навыки кандидата в создании Telegram-ботов с использованием Python и работы с базой данных или хранилищем данных.


## Задача
### Разработать Telegram-бота на Python, который позволяет пользователю
> Сохранять даты дней рождения друзей и получать напоминания о приближении праздника. 

> Бот должен позволять добавлять, удалять и просматривать список дней рождения. 


## Подзадачи
> 1. Работа с пользователем:
– Реализовать команды для добавления новой даты рождения
(например, /add имя дата).
– Реализовать команду для удаления сохранённой даты по имени.
– Реализовать команду для просмотра всех сохранённых дней рождения
пользователя.
– Добавить валидацию введённых данных (например, формат даты,
непустое имя, можно одну).
> 2. Хранение данных:
– Организовать хранение данных пользователя (например, с помощью
SQLite, JSON-файла или другой базы).
– Каждому пользователю хранить собственный список дней рождения.
> 3. Напоминания:
– Реализовать механизм ежедневной проверки и отправки пользователю
напоминаний о приближении дней рождения (например, за 3 дня до
события).
– Формат сообщения должен быть понятным и информативным.
> 4. Обработка ошибок и удобство:
– Корректно обрабатывать ошибки (неправильный формат даты)
– Добавить подсказки и помощь по командам (например, команда /help)
> 5. Архитектура и качество кода:
– Код должен быть структурирован, использовать обработку команд и
сообщений через библиотеку python-telegram-bot или aiogram.
– Обеспечить стабильную работу бота без сбоев.
